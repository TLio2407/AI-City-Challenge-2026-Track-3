\section{VLM-based Extension for PatchCore's Encoder}
\begin{frame}{Opportunity}
\begin{block}{}
    With the rise of Vision-Language Models (CLIP, BLIP, Flamingo, etc), we now have models that can:
    \begin{itemize}
        \item Progress with both visual and linguistic data
        \item Align images with text descriptions
        \item Generalize to novel concepts without fine-tuning or retraining (zero-shot capability).
    \end{itemize}
These models open up a new possibility: anomaly detection that is guided by both visual data and semantic, natural language.
\end{block}
\end{frame}
